# Deployment Directory

This directory contains all deployment-related scripts, configurations, and documentation for the AI-Assisted Crypto Trading System.

## Directory Structure

```
deployment/
├── scripts/      # Deployment scripts (deploy.sh, rollback scripts, etc.)
├── config/       # Environment configurations (test.env, production.env)
├── manifests/    # Deployment manifests with resource identifiers
├── docs/         # Generated documentation and runbooks
└── tests/        # Validation and health check scripts
```
